## 📖 About

The **NEU Library Management System** is a web-based application designed for New Era University to manage library visitors and check-ins. It provides separate interfaces for visitors and administrators, with real-time data storage using Firebase.

### ✨ Features

- **Visitor Check-In** – Visitors can check in by selecting their purpose of visit
- **Admin Dashboard** – Admins can view visitor logs and registered users
- **Google Authentication** – Secure login using Google accounts
- **Real-time Database** – All check-ins are stored in Firebase Firestore
- **Auto Logout** – Visitors are automatically logged out 3 seconds after check-in
- **Role-Based Access** – Admins have special privileges

## 🚀 Getting Started

### For Visitors

1. Go to the website: [https://ephemeral-faun-049a4d.netlify.app](https://ephemeral-faun-049a4d.netlify.app)
2. Click **ENTER**
3. Select **"Visitor"**
4. Sign in with your Google account
5. Complete registration (first time only)
6. Select your reason for visiting
7. Click **CHECK IN**
8. You'll be automatically logged out after 3 seconds

### For Admins

1. Go to the website
2. Click **ENTER**
3. Select **"Admin Server"**
4. Sign in with your authorized admin email:
   - `jcesperanza@neu.edu.ph`
   - `zirkeins.layupan@neu.edu.ph`
5. Complete registration (first time only)
6. Access the admin dashboard where you can:
   - View all visitor check-in logs
   - Search logs by name or reason
   - View registered users
   - See today's check-in statistics
